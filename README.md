# Pelican Plugins

A collection of plugins for [Pelican Panel](https://pelican.dev).

## Available Plugins

| Plugin | Description | Version |
|--------|-------------|---------|
| [PasteFox Share](./pastefox-share) | Share server console logs via pastefox.com | 1.0.0 |
| [Server Folders](./server-folders) | Organize servers into custom folders | 1.0.0 |

## Installation

1. Download the plugin ZIP from the [Releases](https://github.com/FlexKleks/PelicanPlugins/releases) page
2. Go to your Pelican Panel admin area
3. Navigate to **Admin â†’ Plugins**
4. Click **"Import file"**
5. Select the downloaded ZIP file
6. Click **"Import"**

## Plugins

### PasteFox Share

Upload server console logs to [pastefox.com](https://pastefox.com) with one click.

**Features:**
- One-click console log sharing
- Optional API key for extended features
- Configurable visibility, effects, and themes
- Password protection support

[View Documentation](./pastefox-share/README.md)

### Server Folders

Organize your servers into custom folders with live resource monitoring.

**Features:**
- Create custom folders with colors
- Sidebar navigation with server counts
- Live CPU, Memory, and Disk stats
- Multi-language support (EN/DE)

[View Documentation](./server-folders/README.md)
